Query Relative Ratio (QRR) Evaluation Across 113 Queries

QRR measures the relative difference between original and synthetic query costs. For each query, QRR = max(original / synthetic, synthetic / original), ensuring QRR ≥ 1. A value of 1 indicates perfect agreement; larger values indicate greater deviation.
